====================================
测试用全权限 Bearer Token
====================================

Bearer Token (直接复制使用):
Bearer token_1_admin_full_access_test_2024

完整 Authorization Header:
Authorization: Bearer token_1_admin_full_access_test_2024

用户信息:
- User ID: 1
- Email: admin@akcn.com
- Role: Admin
- Team: Platform
- 权限: 全部权限

权限列表:
- applications: [create, read, update, delete]
- subtasks: [create, read, update, delete]
- reports: [create, read, export]
- users: [create, read, update, delete]
- audit: [read, export]
- notifications: [create, read, manage]

使用示例:

1. 获取应用列表:
curl http://localhost:8000/api/v1/applications \
  -H "Authorization: Bearer token_1_admin_full_access_test_2024"

2. 创建新应用:
curl -X POST http://localhost:8000/api/v1/applications \
  -H "Authorization: Bearer token_1_admin_full_access_test_2024" \
  -H "Content-Type: application/json" \
  -d '{"application_name": "Test App", "status": "active"}'

3. 获取当前用户信息:
curl http://localhost:8000/api/v1/auth/me \
  -H "Authorization: Bearer token_1_admin_full_access_test_2024"

4. 计算进度:
curl -X POST http://localhost:8000/api/v1/calculation/calculate \
  -H "Authorization: Bearer token_1_admin_full_access_test_2024" \
  -H "Content-Type: application/json" \
  -d '{"application_ids": [1, 2, 3]}'

前端配置:
在前端代码中设置:
localStorage.setItem('access_token', 'token_1_admin_full_access_test_2024');

或在 axios 配置中:
axios.defaults.headers.common['Authorization'] = 'Bearer token_1_admin_full_access_test_2024';

注意: 这是测试token，生产环境请使用正式的登录流程获取token。
